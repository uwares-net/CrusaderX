TEXTURE FILES

This directory is where texture images for the solar system simulation should be placed.

Required texture files:
- sun.jpg
- mercury.jpg
- venus.jpg
- earth.jpg
- moon.jpg
- mars.jpg
- phobos.jpg
- deimos.jpg
- jupiter.jpg
- io.jpg
- europa.jpg
- ganymede.jpg
- callisto.jpg
- saturn.jpg
- saturn_rings.png
- titan.jpg
- enceladus.jpg
- uranus.jpg
- titania.jpg
- neptune.jpg
- triton.jpg

You can obtain these textures from:

1. NASA's Image Library: https://images.nasa.gov/
2. Solar System Scope: https://www.solarsystemscope.com/textures/
3. Planetpixelemporium: http://planetpixelemporium.com/planets.html

Most of these sites provide free texture maps suitable for educational or personal projects.

If you cannot find the exact texture file for some of the smaller moons, the simulation will fall back to using their defined colors.
